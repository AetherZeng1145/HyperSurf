## 构建和使用
如果您是普通用户，想使用HyperSurf，请去表盘自定义/AstroBox/米坛社区自行搜索下载RPK包安装。但如果您是开发者，您可以克隆该仓库到您的设备上，打开 `AIoT-IDE` 运行 `npm -i` 即可开始构建

## 收费与更新
HyperSurf开发团队承诺————HyperSurf将采取**永久免费+开源**的形式进行更新，不主动向用户收取一分钱
